Set tank configuration parameters.

Parameters
vmaxMaximum volume of the tank in mL.
pmaxMaximum pressure of the tank in Pascal.
vgasGas volume in the tank in cubic mL.

This function sets the tank configuration parameters based on the provided maximum volume (vmax), maximum pressure (pmax), and gas volume (vgas). It calculates the total tank volume (Vtot) in liters and the number of moles of gas (n) using the ideal gas law formula.

Note
The ideal gas constant (R) and temperature (298.15 K) are assumed to be constant. Ensure proper units are used for vmax, pmax, and vgas to get accurate results.

◆ stopControl()

void stopControl ( void )

Stops the control operation.

This function stops the control operation by performing the following actions:

  1. Stops the release of the HB and safety signal.
  2. Resets the PWM.
  3. Releases the DAC.
  4. Sets the speed to 0.
  5. Resets the reference tank level.
  6. Sends a feedback message to the master indicating that the control operation has stopped.
  7. Resets the control active variable.
