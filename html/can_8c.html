void canSend ( uint32_t id,
uint8_t * Txdata )

Sends a CAN message.

Parameters
idThe CAN message identifier.
TxdataPointer to the data to be transmitted.

This function sends a CAN message with the specified ID and data. If there are available transmit mailboxes, the message is added to one of them and sent. If all mailboxes are busy, error handling is triggered.
