void ReloadWatchDog ( void )

Reloads the Independent Watchdog (IWDG) to prevent a system reset.

Return values
None

This function refreshes the IWDG counter to reset the watchdog timer and prevent the system from resetting. It should be called regularly within the watchdog timeout period.

Note
The IWDG must be properly initialized before calling this function.
