void Configuration ( uint32_t id,
uint8_t * data,
mode_t * mode )

Writes customized configuration data to flash memory.

Parameters
IdNode id
DataCAN msg information
Returns
None
Note
  • The configuration must be initialized with CAN_NEWCON.
  • The software responds with an acknowledgment for each message.
  • To make the new configuration available and exit config mode, send CAN_SAVECON.
Attention
The flash memory is completely erased after the command CAN_NEWCON. All customized changes must be written. If there are no customized information the standard will be loaded for this unit

void SetupControlPump ( uint8_t * configuration)

Set up control parameters for the pump based on configuration data.

Parameters
configurationPointer to the configuration data containing control parameters for the pump.

This function extracts control parameters for the pump from the provided configuration data and sets them accordingly. The control parameters include the failure condition, proportional gain (KP), derivative gain (KD), and saturation value. These parameters are expected to be stored at specific offsets defined by the OS_ macros within the configuration data.
