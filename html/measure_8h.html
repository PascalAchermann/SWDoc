Checks if a new measurement of tank pressure is available.

void Measure_Init ( void )

Initializes the measurement module.

This function initializes the measurement module by performing the following tasks:

  • Bringing CS high and then low to initiate communication.
  • Setting up semaphores for data protection for tank pressure, tank temperature, and ambient pressure.
  • Creating a task named "Measure" to handle measurement tasks.

void SetMeasureModeConfig ( uint32_t newmTankPresMode,

Set the measurement modes for tank pressure, tank temperature, and ambient pressure.

Parameters
newmTankPresModeThe new measurement mode for tank pressure.
newmTankTempModeThe new measurement mode for tank temperature.
newmAmbiPresModeThe new measurement mode for ambient pressure.

This function sets the measurement modes for tank pressure, tank temperature, and ambient pressure to the specified values. These modes determine how each parameter is measured. Mode:

  • 0 = CAN
  • 1 to 8 = Analog Channel 1 to 8
  • 9 = Daughterboard
