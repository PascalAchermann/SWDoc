What does the SW?

This software is designed for the STM32L432KCUX

  • install 'Espressif IDF' extension from Expressif Systems
  • launch code from the project root folder
  • select Kit (might need to rescan for kits), Eelect GCC 12.2.0. xtensa-esp32-elf
  • bottom toolbar: select COM port and project
    • ESP32-WROOM-32E, 4 MB Flash
    • ESP32, ESP32 chip via ESP-Prog

Flash via JTAG:

  • config -> security featues -> set Flash to 4 MB
  • select JTAG (Flash)
  • run UsbDriverTool: WinUSB for USB Verbundgerät, VID 0403, DID 6010
  • run OpenOCD (bottom toolbar)

Debug: https://github.com/espressif/vscode-esp-idf-extension/blob/master/docs/tutorial/debugging.md

  • Debug toolbar, create launch.json, select ESP-IDF type

Troubleshooting

  • Program upload failure
    • Hardware connection is not correct: run idf.py -p PORT monitor, and reboot your board to see if there are any output logs.
    • The baud rate for downloading is too high: lower your baud rate in the menuconfig menu, and try again.
