◆ RunningMode()

static void RunningMode ( void )
static

Run the system in the main operational mode.

This function is responsible for controlling the main operational mode of the system. It performs various tasks based on the current sensor readings and control settings.

The function verifies the validity of sensor readings from ambient pressure, tank pressure, tank temperature, and reference tank values. If all sensor values are non-zero, the function proceeds to determine the appropriate control action based on the current system state and control settings.

The control action is determined by the enControl() function, which evaluates the system state and returns an integer value indicating the desired action:

  • Case 1: Close the reflow valve, load the capacitor array if ready, and adjust the pump speed based on control parameters.
  • Case 2: Close the pump valve, discharge the capacitor array, stop the pump, and initiate the reflow process based on control parameters.
  • Case 3: Close the reflow valve, load the capacitor array if ready, and adjust the pump speed based on control parameters.
  • Default: Close both the pump and reflow valves and stop the control process.
Note
This function assumes that all necessary sensor values and control settings are properly initialized and available for use. Additionally, it relies on the proper functioning of auxiliary functions such as getAmbiPres(), getTankPres(), getTankTemp(), refTank(), CapacitorArrayReady(), setDigHB(), controlpump(), DischargeCapacitor(), enControl(), and stopControl().

◆ stopControl()

void stopControl ( void )

Stops the control operation.

This function stops the control operation by performing the following actions:

  1. Stops the release of the HB and safety signal.
  2. Resets the PWM.
  3. Releases the DAC.
  4. Sets the speed to 0.
  5. Resets the reference tank level.
  6. Sends a feedback message to the master indicating that the control operation has stopped.
  7. Resets the control active variable.
